# almasiah-display (GitHub Pages)

موقع بسيط لعرض المحتوى على الشاشات (Digital Signage) مع هوية الماسية.

## التشغيل
1) ارفع الملفات إلى مستودع GitHub.
2) فعّل GitHub Pages من Settings > Pages واختر Branch: main و Folder: /root
3) افتح الرابط على الشاشات.

## التحكم
- الوضع الافتراضي: playlist (يتنقل بين الأقسام)
- أو افتح:
  - ?mode=ads
  - ?mode=numbers
  - ?mode=live
  - ?mode=static

## تعديل المحتوى
عدّل الملف: data/content.json ثم Commit.
